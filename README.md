# Орфоэпический тренажер

Веб-приложение для тренировки правильного ударения в словах. Помогает подготовиться к экзаменам (ЕГЭ/ОГЭ) или просто повысить культуру речи.

## Особенности
- **Интерактивный тренажер:** JS-скрипт мгновенно проверяет правильность ударения при клике на гласную букву и подкрашивает результат.
- **Полный словарь:** Отдельная страница со всеми словами, которые используются в приложении.

## Технологии
- Python 3.x
- Django 5.x
- JavaScript (Vanilla JS)
- SQLite (база данных)

## База данных и словарь
В проекте используется упрощенная схема:
- **Системные данные** (админка, сессии): хранятся в SQLite.
- **Словарь тренажера:** хранится в статичном JSON-файле в "OrthoepicalTest/static/OrthoepicalDictionary.json", что позволяет приложению работать мгновенно без настройки тяжелых баз данных.

## Как запустить проект у себя

Чтобы запустить тренажер локально, выполните следующие шаги:

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com
   cd название-репозитория
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для Linux/macOS:
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Выполните миграции базы данных:**
   *Это необходимо для сессий*
   ```bash
   python manage.py migrate
   ```

5. **Запустите сервер:**
   ```bash
   python manage.py runserver
   ```

Теперь приложение доступно по адресу: `http://127.0.0`

## Использование
- Перейдите на страницу тренажера.
- Кликайте на гласные буквы в словах.
- Зеленый цвет — ударение верное, красный — ошибка.
- Используйте страницу словаря для поиска конкретных слов и их изучения.
